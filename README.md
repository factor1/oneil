# O'Neil Printing
A custom WordPress theme developed by Factor1 and designed by Rule29.

## Packages/Tools Used
We use npm and gulp in addition to these dependencies:
```json

"ginger-grid": "^2.0.1",

"waypoints": "^4.0.1",
"share-button": "^1.0.3",
"moment": "^2.16.0"
```
